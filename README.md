# EliteQR Frontend

A starter React + Vite front-end for creating and previewing QR codes.

## Setup

1. Open a terminal in `c:\Users\vasav\Desktop\projects\eliteqr`
2. Run `npm install`
3. Run `npm run dev`

## What this includes

- React + TypeScript app
- QR code preview with the `qrcode` package
- Local save list stored in `localStorage`
- Simple form for URL and QR label

Next step: add backend integration for dynamic redirect URLs and analytics.
